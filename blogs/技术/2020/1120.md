## 和前端开发相关的项目角色有哪些？

      PM产品经理
      UE视觉设计师
      FE前端开发
      RD后端开发
      CRD移动端开发
      QA测试人员

## 一个完整的项目要分哪些阶段？

      需求分析——>技术方案设计——>开发——>联调——>测试——>上线

## 评审项目需求时需要注意哪些事项？

      了解背景
      质疑需求是否合理
      需求是否闭环
      开发难度如何
      是否需要其他支持
      不要急于给排期

## 如何做好技术方案设计？

      求简,不过渡设计
      产出文档
      找准设计重点
      组内评审
      和RD、CRD沟通
      发出会议结论

## 如何保证代码质量？

      如何反馈排期
      符合开发规范
      写出开发文档
      及时单元测试
      Mock API
      Code Review

## 项目过程中 PM 加需求怎么办？

      不能拒绝,走需求变更流程即可
      如果公司有规定,则按规定走
      否则,发起项目组和leader的评审,重新评估排期
